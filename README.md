# customer-shopping-behavior-analysis
Customer shopping behavior analysis using Python, SQL, and Power BI to generate business insights.

## Project Overview

This project analyzes customer shopping behavior to uncover purchasing patterns and generate actionable business insights.
The goal of this project is to demonstrate an end-to-end data analysis workflow including **data cleaning, data analysis, and data visualization**.

## Problem Statement

Businesses need to understand customer purchasing behavior to improve marketing strategies, product placement, and sales performance.
This project analyzes shopping data to identify key trends and customer spending patterns.

## Dataset

The dataset contains customer shopping information including:

* Customer ID
* Age
* Gender
* Product Category
* Purchase Amount
* Payment Method
* Purchase Frequency

## Tools & Technologies

* **Python** – Data cleaning and preprocessing (Pandas)
* **MySQL** – Data analysis using SQL queries
* **Power BI** – Data visualization and dashboard creation

## Project Workflow

1. **Data Cleaning (Python)**

   * Handled missing values
   * Removed duplicates
   * Prepared data for analysis

2. **Data Analysis (SQL)**

   * Calculated total revenue
   * Identified top spending customers
   * Analyzed product category performance
   * Generated customer behavior insights

3. **Data Visualization (Power BI)**

   * Created interactive dashboard
   * Visualized sales trends
   * Highlighted key insights

## Key Insights

* Identified the **highest revenue-generating product categories**
* Found **customer segments with the highest spending**
* Observed **purchase trends across different demographics**
* Analyzed **payment method preferences**
